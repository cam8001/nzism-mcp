# Data files

These files can be downloaded and generated at build time, but are included here for convenience.

- `nzism.xml` — NZISM V.3.9, November 2025. Downloaded 24 April 2026 from https://nzism.gcsb.govt.nz/ism-document/xml
- `deeplinks.json` — anchor ID mapping scraped from the NZISM HTML document
- `nzism_index.json` — pre-built search index, generated by `build_index.py`

To regenerate the index after updating the XML:

```bash
uv run python build_index.py
```
